# Task Management App

A full-stack task management application for creating, organizing, and tracking tasks with intuitive features and drag-and-drop support. Built with a modern tech stack and designed for smooth local development and deployment.

## 🚀 Features

✨ **Core Functionality**
- Create, edit, and delete tasks  
- Drag and drop tasks between columns  
- Filter and search tasks by status or title  
- Responsive UI for desktop & mobile

✨ **Modern Tools**
- Frontend and backend separation  
- Docker support for easy local setup  
- RESTful API for task management

## 🧠 Tech Stack

### Frontend
- React (with TypeScript)  
- Tailwind CSS / UI framework  
- Axios for API requests

### Backend
- Node.js + Express.js  
- REST API  
- MongoDB (database)  
- Docker support

### TEST (ONGOING)
- JEST
- SUPERTEST

## 🔧 Prerequisites

Before running locally, make sure you have the following installed:

- **Node.js** (v16+)  
- **Docker & Docker Compose** (optional but recommended)  
- **Git**
